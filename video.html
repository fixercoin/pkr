A trilogia Front-End WEB

Estas são as três linguagens que qualquer um que queira desenvolver para web precisa dominar.

HTML

HTML (HyperText Markup Language) é o bloco de construção básico da web e é utilizado para definir a estrutura e o significado do conteúdo.

CSS

CSS (Cascading Style Sheet) é usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

JavaScript

JavaScript (JS) é uma linguagem de programação usada para criar páginas web e aplicativos interativos. É uma das principais tecnologias da web, juntamente com o HTML e o CSS.
